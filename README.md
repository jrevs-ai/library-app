# The Library

A personal book catalogue — React + Supabase + Vercel.

**To get it running**, read [SETUP.md](./SETUP.md). It's a ~20-minute walkthrough.

## What's here

- `src/App.jsx` — the main app component
- `src/supabase.js` — Supabase client setup
- `seed.sql` — run this once in Supabase's SQL editor to create the table and load 319 books
- `.env.example` — template for environment variables (needed for local dev only)

## Categories

The books are organised into:
- Business & Investing
- History
- Military History
- Biography & Memoir
- Science & Ideas
- Narrative Non-Fiction
- Fiction
- Literature & Reference
- Self-Development
- Australia

## Stack

- **Frontend**: React 18 + Vite + Tailwind + Lucide icons
- **Database**: Supabase (Postgres)
- **Hosting**: Vercel
- **Total cost**: $0/month at personal scale
